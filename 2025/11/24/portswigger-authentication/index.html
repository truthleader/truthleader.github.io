基于密码的认证机制

用户名

  1. 查找潜在公开的用户名
    1. profile，可能profile里的名字是用户名
    2. http报文里的email，可能包含管理员的邮箱，而邮箱可能包含管理员的用户名
  2. 判断用户名的recognizable pattern
  3. 尝试 username enumeration
    1. 检查status code
    2. 检查 error 信息
    3. 检查 response times

密码

  1. 检查密码policy
  2. 根据policy，用户习惯等准备字典
  3. 爆破
